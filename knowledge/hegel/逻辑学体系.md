## 体系全景

黑格尔逻辑学 = **纯粹概念的自我运动**。它不是形式逻辑（亚里士多德），也不是先验逻辑（康德），而是**概念自己展开为自己的内容**——逻辑学就是形而上学。

### 三层结构

```
                         ┌─ 生命 (Life)
                         │
                     ┌─ 理念 ─┼─ 认识的理念
                     │   │     │   ├─ 真的理念（理论）
                     │   │     │   └─ 善的理念（实践）
                     │   │     │
                     │   └─── 绝对理念 (Absolute Idea)
                     │
         ┌─ 概念论 ─┼─── 客体
         │   (主观    │     ├─ 机械性
         │    逻辑)   │     ├─ 化学性
         │           │     └─ 目的论
         │           │
         │           └─── 主体（概念本身）
         │                 ├─ 概念 (Concept)
         │                 ├─ 判断 (Judgment)
         │                 └─ 推论 (Syllogism)
         │
         │           ┌─ 现实 (Actuality)
         │           │   ├─ 实体关系
         │           │   ├─ 因果关系
         │           │   └─ 相互作用
         │           │
逻辑学 ─┼─ 本质论 ─┼─── 现象 (Appearance)
         │   (客观    │     ├─ 现象世界
         │    逻辑)   │     ├─ 内容与形式
         │           │     └─ 关系
         │           │
         │           └─── 本质作为反思 (Essence as Reflection)
         │                 ├─ 同一、区别、矛盾
         │                 ├─ 本质的根据
         │                 └─ 实存 (Existence)
         │
         │           ┌─ 尺度 (Measure)
         │           │
         └─ 存在论 ─┼─── 量 (Quantity)
                     │     ├─ 纯量
                     │     ├─ 定量
                     │     └─ 程度
                     │
                     └─── 质 (Quality)
                           ├─ 存在 (Being)
                           ├─ 定在 (Dasein)
                           └─ 自为存在 (Being-for-self)

起点：纯存在 (Pure Being) → 无 (Nothing) → 变 (Becoming)

终点：绝对理念 —— 概念在其自身中把握自己的全部内容，方法回到自身
```

### 核心理解要点

1. **起点 = "纯存在"**：不预设任何规定性，最空洞、最直接的东西。但这个空洞本身就等于"无"——而"存在"与"无"之间的运动就是"变"（生成）。整个逻辑学就是从"变"开始概念自我规定的展开。

2. **"辩证法"不是方法，是事情本身**：不是黑格尔"用"辩证法来分析东西；而是概念自身的内在矛盾迫使它超出自身、走向更丰富的规定。辩证法 = 概念本身的生命节奏。

3. **肯定 → 否定 → 否定之否定**（正题→反题→合题 这种说法不准确，黑格尔自己不用这个公式）。每个范畴因其内在的不充分性而过渡到它的对立面，最终在更高层次上统一。

4. **存在论 → 本质论 → 概念论 的内在逻辑**：
   - **存在论**：范畴"过渡"（一个变成另一个）——质的逻辑
   - **本质论**：范畴成对"映现"（互相照亮，表层/深层）——关系的逻辑
   - **概念论**：概念"发展"自身——生命的逻辑

5. **《逻辑学》和黑格尔整个体系的关系**：
   - 逻辑学 = 理念在自身中（纯概念）
   - 自然哲学 = 理念的外化（概念在空间中外在存在）
   - 精神哲学 = 理念回到自身（概念在人的历史与意识中自觉）

## 建议学习路径

### 第一阶段：概览（2-3 周）
通读《小逻辑》，不求逐段理解，重点把握：
- 全书的三层结构是什么
- 为什么起点必须是"纯存在"
- 绝对理念为什么不是终点而是"回到起点"

### 第二阶段：逐卷精读（按你的节奏）
从大逻辑的"存在论"开始，每读一节：
1. 先用自己的话复述黑格尔的推导步骤
2. 找到关键概念对立（如 质/量、同一/区别/矛盾）
3. 问自己：为什么前一个范畴"逼出"了后一个？逼出的动力在哪？
4. 对照张世英的解读检验理解

### 第三阶段：专题打通
把逻辑学与黑格尔体系的其余部分联系起来：
- 《精神现象学》作为"逻辑学的导论"
- 自然哲学 / 精神哲学 是否真的从逻辑范畴推出来
- 马克思、克尔凯郭尔、海德格尔对黑格尔的批判是否打中要害
